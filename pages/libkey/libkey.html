libkey

A C library developed for platforms that support the ncurses terminal handling library.
Its purpose is to make handling the keyboard inside of a terminal easier by giving the
programmer the tools needed to emulate the modal behavior of editors like Vi and Vim.

The library has interfaces for creating modes, and attaching sequences of keys that can
execute callbacks upon their completion to the modes. Key sequences can be prefixed with
a number to tell the callback how many times to execute. Callbacks can also accept an
arbitrary number of 'argument' keys, which allows the programmer to simulate behavior
similar to the motions, marking, and register features found in Vi and Vim.

The project follows a couple main design principles.
  • Portability (100% ANSI C compliance)
  • Being tested to hell and back
  • Being stackful (no calls to malloc and friends)
  • Be designed around single-threaded applications
